Welcome to the CrimeAlert Machine Learning API

Get Crime Predictions for the City of London:


HTTP Method: /predict (GET Request)

Content Type: application/json

Returns: JSON Object with crime prediction, probability of the prediction of equivalent class, all crime classes

Field Type Description
month Integer (REQUIRED) The month to predict crime (i.e January = 1 - December = 12)
lon Float (REQUIRED) The Longitude of the Location to predict in the City of London (i.e -0.079786)
lat Float (REQUIRED) The Latitude of the Location to predict in the City of London (i.e 51.517576)

Example Request:

/predict?month=2&lon=-0.079786&lat=51.517576
